# BUM

## Installation

1. Set `LEAN_HOME` environment variable with the path to Lean 4 installation:

```
$ echo 'export LEAN_HOME=/path/to/lean4 >> ~/.bashrc'
```

2. Run Make:

```
$ make
```

3. And copy `bum` binary to your preferred place:

```
$ sudo cp bum /usr/local/bin
# cp bum ~/.local/bin
```
