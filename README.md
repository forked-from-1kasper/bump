# BUMP

## Installation

1. Set `LEAN_HOME` environment variable with the path to Lean 4 installation:

```
$ echo 'export LEAN_HOME=/path/to/lean4 >> ~/.bashrc'
```

2. Run Make:

```
$ make
```

3. And copy `bump` binary to your prefer`red place:

```
$ sudo cp bump /usr/local/bin
# cp bump ~/.local/bin
```
